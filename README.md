# wx-english-dictation
## 项目介绍
EnglishDictation 是一个用于英语听写练习的小程序，帮助学习者通过听写提高英语听力与拼写能力。项目基于微信小程序云开发，拥有音频播放、实时听写输入、答案校验及错题收藏等功能，适合从入门到中高级水平的英语学习者。

基于微信小程序云开发和 Taro 框架的英文听写练习应用。

## 功能特性

- 音频播放：播放英文句子音频
- 听写输入：用户输入听到的英文内容
- 答案比对：比较用户输入与正确答案
- 云数据存储：句子和音频数据存储在云数据库
